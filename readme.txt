== Description ==

This is a Plugin that shows your recommendations from QUOTE.fm.

== Installation ==

How to install the plugin:

1. Upload `quotefm.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [quotefm username="your_quotefm_username"] in your site.
